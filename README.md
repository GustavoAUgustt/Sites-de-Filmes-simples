# 🎬 Universo dos Filmes

Um site simples de catálogo de filmes desenvolvido com **HTML**, **CSS** e **JavaScript**.

O projeto foi criado com o objetivo de praticar conceitos básicos de desenvolvimento Front-End, como estruturação de páginas, estilização e interatividade.

---

# 📸 Preview

O site possui:

* Cards de filmes
* Pôsteres reais
* Botões interativos
* Tema escuro inspirado em plataformas de streaming

---

# 🚀 Tecnologias Utilizadas

* HTML
* CSS3
* JavaScript Basico

---

# 📂 Estrutura do Projeto

```txt
site-filmes/
│
├── index.html
├── style.css
└── script.js
```

---

# ⚙️ Como Executar

## 1. baixe o projeto

apenas baixe os arquivos manualmente.

---

## 2. Abra a pasta do projeto

Verifique se os arquivos estão organizados corretamente:

```txt
index.html
style.css
script.js
```

---

## 3. Execute o projeto

Basta abrir o arquivo:

```txt
index.html
```

em qualquer navegador.
